## 🎯 Features

Here's what you can do in the Weather Application:

- **Real-Time Weather**: View current weather conditions including temperature, humidity, wind speed, and visibility.
- **Hourly Forecast**: Check upcoming hourly weather updates for better planning.
- **7-Day Forecast**: Get a complete weekly weather forecast.
- **Air Quality Information**: Monitor air quality and environmental conditions.
- **Interactive Weather Map**: Explore weather locations using an interactive map.
- **Location Search**: Search weather information for different cities.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.

---

## 🪜 The Process

I started by designing the frontend to focus on creating a clean and modern dashboard that presents weather information in an easy-to-read layout.

Once the user interface was complete, I developed the backend using ASP.NET Core to handle weather requests and communicate with external weather APIs.

One of the biggest challenges was organizing different types of weather data—current conditions, hourly forecasts, weekly forecasts, and air quality—into a single responsive interface without overwhelming the user.

After integrating the APIs, I focused on improving performance, handling loading states, managing API errors gracefully, and ensuring the application remained responsive across different screen sizes.

## 🚀 Running the Project

### Step 1 — Clone the Repository

```bash
git clone https://github.com/khanyasser2006/weather-application.git
```

---

### Step 2 — Run the Backend

**Prerequisites:** .NET 8 SDK

```bash
cd backend
dotnet restore
dotnet run
```

---

### Step 3 — Run the Frontend

**Prerequisites:** Node.js 18+

```bash
cd frontend
npm install
npm run dev
```

---

### Step 4 — Open the Application

Open the address shown in your terminal (usually):

```
http://localhost:5173
```
